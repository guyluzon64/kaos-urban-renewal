# מדריך שימוש קצר

## מה האתר עושה?

האתר מאפשר לחפש עיר, רחוב, מתחם או מספר תכנית ולראות איזה מידע ציבורי
נמצא, מה השלב התיאורי שניתן לזהות, ומאיפה הגיע המידע.

## מה האתר לא עושה?

- אינו נותן ייעוץ משפטי, תכנוני, שמאי או נדל״ני.
- אינו חוזה אם פרויקט יתקדם.
- אינו משלים מידע חסר.
- אינו מחליף עיון במסמכי המקור או פנייה לגורם מקצועי מתאים.

## הפעלה

```powershell
python scripts/update_public_data.py
streamlit run app/urban_renewal_legal_scout.py
```

## קריאת התוצאות

הכיתוב `שלב ציבורי מזוהה: X מתוך 9` מסכם את החיווי שנמצא ברשומות
הציבוריות. פתחו את כפתור המקור הרשמי כדי לקרוא את הפרטים, המסמכים
והתאריכים במאגר המקורי.

מידע חסר נשאר חסר. גם תאריך רענון חדש של המערכת אינו מבטיח שכל פרט בתוך
המקור עודכן באותו יום.
